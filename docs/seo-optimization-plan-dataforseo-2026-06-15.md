## SEO Strategy

### Recommended Positioning

Keep the approved promise:

> Affordable custom websites, live in 48 hours.

Add SEO support around it:

- Affordable website design for small businesses in Niagara and Toronto
- One-page websites for local businesses that need calls, bookings, visits, and quote requests
- Fast custom websites for businesses with real services, reviews, photos, hours, and locations

Do not reposition as a generic web design agency. The site should look like the fastest practical path for owner-operated businesses that need a real site.

## Execution Plan

### Phase 1: Technical SEO Foundation

Goal: make the existing site easier to index and understand.

Implement:

- Add `robots.txt`
- Add `sitemap.xml`
- Add canonical URL on the homepage
- Add Open Graph and Twitter metadata
- Add `Organization` or `LocalBusiness` structured data
- Add `Service` structured data for website design services
- Add image width/height attributes where practical
- Confirm the Vercel production domain resolves cleanly to one canonical URL

Recommended homepage title:

`Affordable Custom Websites for Local Businesses | Well and Good Websites`

Recommended homepage meta description:

`Affordable custom one-page websites for local businesses in Niagara and Toronto. Well and Good Websites builds fast, practical sites designed to help customers call, book, visit, or request a quote.`

### Phase 2: Homepage SEO Copy Pass

Goal: keep the current brand/copy intact while making the page more search-legible.

Add or adjust copy so the homepage naturally includes:

- affordable website design
- small business websites
- one-page websites
- Niagara
- Toronto
- web design for local businesses
- calls, bookings, quote requests, directions, hours, reviews

Specific changes:

- Add a concise intro paragraph under the hero proof line using "affordable website design for small businesses in Niagara and Toronto."
- Expand the Services section with a plain-language "What is included" paragraph.
- Add a short FAQ section near the bottom.
- Add internal anchors for future service/location pages.

Suggested FAQ topics:

- How much does a small business website cost?
- Can a one-page website rank on Google?
- What do you need from me to launch in 48 hours?
- Do you help with Google Business Profile?
- Is this better than Wix or Squarespace for my business?

### Phase 3: Add Priority Landing Pages

Goal: avoid forcing every keyword onto the homepage.

Create these pages in order:

1. `/web-design-niagara/`
   - Primary keyword: `web design niagara`
   - Supporting terms: `web design niagara falls`, `website design niagara`, `small business websites niagara`
   - Include Niagara-specific businesses: restaurants, trades, salons, barbers, auto shops, contractors, artists.

2. `/affordable-website-design/`
   - Primary keyword: `affordable website design`
   - Supporting terms: `affordable website design for small business`, `cheap website designers`, `website packages pricing`
   - Position carefully: affordable, not low-quality.

3. `/one-page-websites/`
   - Primary keyword: `one page website design`
   - Supporting terms: `single page website design`, `single page websites`, `one page website examples`
   - Explain when a one-page site is enough and when it is not.

4. `/website-design-toronto-prices/`
   - Primary keyword: `website design toronto prices`
   - Use later, after the first three pages are live.
   - This can compare the $497/$997/$1,497 packages against typical agency timelines and pricing.

### Phase 4: Local SEO Signals

Goal: make "near me" and local-pack opportunities possible.

Implement outside the codebase:

- Create or optimize a Google Business Profile if eligible.
- Use a consistent business name, website URL, phone/contact method, service area, and description.
- Add Well and Good Websites to relevant Canadian/local directories only where quality is reasonable.
- Ask each completed client for a review that mentions the concrete work performed.
- Add case-study links from completed client sites back to Well and Good where appropriate.

Suggested review language to request naturally:

- custom website
- local business website
- fast launch
- helped customers call/book/find us
- Niagara or Toronto if true

### Phase 5: Content and Proof

Goal: build authority without writing generic SEO filler.

Create compact case studies for:

- FrankBaggetta.ca
- Evelyn's Sandwich Factory

Each case study should include:

- Business type
- Problem before the site
- What the one-page site includes
- Screenshots
- Launch timeline
- Calls/bookings/location/reviews sections
- Link to live site

Add one practical article later:

`How much does a small business website cost in Ontario?`

This supports the pricing searches without bloating the homepage.

## Plain-English Recommendation

The safest SEO path is:

1. Fix the crawl/indexing basics.
2. Keep the homepage promise intact.
3. Add one strong Niagara page first.
4. Add one affordable-website-design page second.
5. Add one one-page-websites page third.
6. Treat Toronto as a longer-term target, not the first fight.

This gives the site a realistic chance to rank for smaller commercial searches before competing with established Toronto agencies.
